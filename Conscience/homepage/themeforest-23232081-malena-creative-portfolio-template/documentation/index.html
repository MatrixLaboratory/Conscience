C) CSS Files and Structure - top

Styles

We're using 9 core CSS files in this template. we have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

fontawesome-all.min.css

Base styles for FontAwesome icon fonts.

ionicons.min.css

Base styles for IconIcos icon fonts.

bootstrap.min.css

This Stylesheet contains responsive grid system, default element style based on bootstrap. It is bootstrap framework style css file.

stylesheet.css

Use for OpenSans font

swiper.min.css
jquery.fullpage.min.css

Use for scrolling.

vegas.min.css

Use for slideshow

Main css file where you can do own customization
